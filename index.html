Ragial Notifier

An Android Application developed to act as a convenience for ragial.com, an item price tracking website for the popular MMORPG Ragnarok Online, developed by Gravity Inc. The entire application is developed with efficiently delivering notifications to users when an item they are tracking is on sale. This application is also very efficient when parsing ragial.com, so as to reduce the load on their servers.

Basic Building Blocks

A mixture of various building blocks for apps. Contains the sort folder, which holds the ParallelQuickSort class. This class manages a ThreadPool to use multiple threads and sort the data using QuickSort algorithm as its base. Generally 2-3 times faster than Arrays.sort(), and while marginally slower than Arrays.parallelSort(), it can sort up to 800 Million data items without crashing. Arrays.parallelSort() will crash on any data set larger than 100 Million data items.

Link : http://titu1994.github.io/Basic-Blocks/

Link to ParallelQuickSort : https://github.com/titu1994/Basic-Blocks/blob/master/Basic%20Blocks/src/com/somshubra/sort/ParallelQuickSort.java
